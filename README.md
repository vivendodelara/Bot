# 🤖 Bot BDFD

Um bot Discord criado com BDFD (Bot Discord For Discord).

## 📋 Comandos Disponíveis

### `/ping`
- **Descrição:** Verifica a latência do bot
- **Uso:** `/ping`
- **Resposta:** Mostra a latência do bot em millisegundos

## 📁 Estrutura

```
Bot/
├── commands/
│   └── ping.bdfd    # Comando de ping
└── README.md        # Este arquivo
```

## 🚀 Como Usar

1. Copie os arquivos dos comandos para seu bot BDFD
2. Configure o prefixo desejado
3. Use `/ping` para testar a conexão
